# ThirdPersonShooterDemo

## Description
A very simple Third-Person Shooter demo made using Unreal Engine 4 and C++.
Simple level that restarts if all enemies are dead or player dies.

Currently in early-developing stages.

Utilizes free assets obtained from the UE4 Marketplace:
- https://www.unrealengine.com/marketplace/en-US/product/paragon-wraith
