# BangingBots

## Description
A simple Third-Person Shooter game made using Unreal Engine 4 and C++.

Currently in early-developing stages.

Wave-based combat as the player takes on waves of Robots.

Utilizes free assets obtained from the UE4 Marketplace:
- https://www.unrealengine.com/marketplace/en-US/product/paragon-wraith
- https://www.unrealengine.com/marketplace/en-US/product/fps-weapon-bundle
